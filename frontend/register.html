HTTP Action: POST /register/

Registration triggers a POST request sending a JSON body with the username and password to our new custom endpoint.

Password Hashing

On the backend, Django never stores passwords in plain text. The serializer intercepts the request and calls User.objects.create_user(), which hashes the password using PBKDF2/SHA256 automatically.

Security Note

In standard REST architectures, the password field is marked as write_only=True in the serializer. This ensures that the hashed or plain passwords are never returned in response payloads.
